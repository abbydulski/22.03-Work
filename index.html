1. Introduction

  • Tasked with improving the current data logger system used by the Blackfeet Indian Reservation in northern Montana
  • The reservation uses the data loggers for research purposes
  • Data loggers are typically set up facing each other 15 ft apart
  • Components of the current logger are rebar, a camera trap, an audio recorder, and batteries
  • Current system is inefficient and laborous
  • Using solar panels, a voltage booster, and soldering a cutting of the components wires we were able to come up with a new system resolving many of the complications of the current one
  • New system components are rebar, a camera trap, an audio recorder, a solar panel pack, and a portable charger
